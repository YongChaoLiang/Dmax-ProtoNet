# Dmax Prediction Dataset

This folder contains the datasets used for predicting the critical casting diameter \(D_{max}\) of bulk metallic glasses.

The folder includes:

- One original training dataset
- One original test dataset
- Three augmented training datasets

These files are mainly used to compare the effects of different data augmentation strategies on \(D_{max}\) prediction performance.

## File Description

| File name | Description |
|---|---|
| `train_data2.csv` | Original training dataset. This file contains the original alloy composition features and the target value `Dmax`. It is used for baseline model training and comparison. |
| `test_data2.csv` | Original test dataset. This file is used only for final model evaluation. It should not be used for data augmentation, model training, feature selection, or hyperparameter tuning. |
| `gn_train_data2.csv` | Augmented training dataset generated by a Gaussian-noise-based augmentation method. This file is used to evaluate the effect of Gaussian noise augmentation on model performance. |
| `mixup_train_data2.csv` | Augmented training dataset generated by the Mixup method. Synthetic samples are created by interpolating between existing training samples and their corresponding `Dmax` values. |
| `smogn_train_data2.csv` | Augmented training dataset generated by the SMOGN method. This file is mainly used to address imbalance in regression tasks, especially to improve learning in underrepresented high- or low-`Dmax` regions. |

## Dataset Usage

The recommended usage is as follows:

1. Use `train_data2.csv` as the original training dataset to obtain the baseline model performance.
2. Use `gn_train_data2.csv`, `mixup_train_data2.csv`, and `smogn_train_data2.csv` separately as augmented training datasets to compare different data augmentation strategies.
3. Use the same `test_data2.csv` for final evaluation of all models.
4. Do not merge the test dataset into the training dataset.
5. Do not use the test dataset for data augmentation, model training, feature selection, or parameter selection.

## Target Variable

The prediction target is:

```text
Dmax
```

`Dmax` represents the critical casting diameter of bulk metallic glasses and is commonly used to evaluate the glass-forming ability of alloys.
